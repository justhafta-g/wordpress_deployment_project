PHP-FPM role 
=========

Instaling php7.4-fpm

Requirements
------------
Packeges will be instaled:

  - python-mysqldb
  - libcurl3
  - php{{ php_version }}
  - php{{ php_version }}-fpm
  - php{{ php_version }}-mysql
  - php{{ php_version }}-xml
  - php{{ php_version }}-curl
  - php{{ php_version }}-gd
  - php{{ php_version }}-mbstring
  - php{{ php_version }}-zip
